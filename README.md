# odin-recipes

This is the first project in the foundations course for the odin project. I will be creating a website that has a main index page with links to a few recipes.

 I believe I have completed the project with all the requirements. It was very satisfying to put into practiced everything I have learned so far like navigating the ubuntu terminal, creating ordered and unordered lists, using all the different tags, elements and attributes.

 I hit some challenges with the lists. They weren't showing up properly in the browser. After going back and looking at the lesson for lists. I found out that I had wrote the tags in the wrong order. I put li first followed by adding the items in the list in the ul or ol tags.

 There were some things I forgot to do when adding the attribution link for the recipe pictures. I had not added the target and rel attributes to the anchor tag. I only realized I forgot about them when I went back looking through the other lessons to find how to fix the lists. Which was lucky otherwise I wouldn't have remembered at all to add those attributes.

 Overall I'm satisfied with the result of this project and I can't wait to keep going through the lessons and improving my skills in web development.
